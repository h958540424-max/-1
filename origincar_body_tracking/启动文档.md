小车控制指令
开启相机节点：
ros2 launch origincar_bringup usb_websocket_display.launch.py       //已在运动控制中包含

开启模型处理：
ros2 launch racing_obstacle_detection_yolo racing_obstacle_detection_yolo

开启运动控制：(相机启动节点和二维码识别节点已包含在内)
ros2 launch origincar_competition start.launch.py

开启底盘节点:
ros2 launch origincar_base origincar_bringup.launch.py    

开启上位机节点：
ros2 launch rosbridge_server rosbridge_websocket_launch.xml 

大模型人像识别节点：
ros2 run image_upload_analyzer image_upload_analyzer
需安装火山功能包库，使用pip install -U volcengine-python-sdk[ark]安装
相较于之前版本的模型识别，移除tos，直接将图像编码为JPEG格式的字节流，在将字节流编码为base64字符串,使用URl方案传递

二维码识别节点：
ros2 run origincar_qr li4       //已在运动控制中包含
需安装pyzbar库，使用pip install pyzbar安装
